# OLA Rides Analytics Dashboard

## Project Overview

This project focuses on analyzing OLA ride booking data using Power BI to identify booking trends, cancellation patterns, customer demand behavior, operational performance, and revenue insights.

The dashboard provides interactive KPI reporting and business intelligence visualizations supporting data-driven decision-making.

---

# Business Problem

Ride-booking platforms generate large volumes of operational data daily. Businesses require analytical dashboards to monitor:

- Ride demand
- Revenue performance
- Booking trends
- Cancellation patterns
- Peak booking hours
- City-wise ride performance
- Driver efficiency

This project transforms raw operational ride-booking data into actionable business insights using Power BI.

# Key Insights

- Peak ride demand occurred during evening hours.
- Certain cities generated significantly higher booking volumes.
- Cancellation rates increased during high-demand periods.
- Premium ride categories contributed higher average revenue.
- Operational inefficiencies were identified through cancellation trend analysis.

---

# Business Recommendations

- Increase driver allocation during peak demand hours.
- Improve operational efficiency to reduce ride cancellations.
- Focus business strategies on high-performing cities.
- Monitor cancellation trends to improve customer experience.
